# 📚 AI-Based Study Planner

An AI-powered Study Planner web application that helps students organize their study schedule, prioritize subjects based on exam dates and difficulty, track study progress, and improve productivity using a Pomodoro Timer.

---

## 🚀 Features

* 🔐 Student Login
* 📚 Add and Manage Subjects
* 🤖 AI-Based Study Recommendation
* 📅 Study Planner
* 📊 Progress Tracker
* ⏱️ Pomodoro Timer
* 🔔 Study Reminder
* 🌙 Responsive User Interface
* 💾 Local Storage Support

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## ⚙️ How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension (if not already installed).
4. Right-click **login.html**.
5. Select **Open with Live Server**.
6. Start using the AI Study Planner.

---

## 🤖 AI Logic

The application recommends the most important subject by considering:

* Subject Difficulty
* Exam Date
* Remaining Days

The subject with the highest priority is recommended for study first.
